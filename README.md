## About

I'm **Dharmik Patel**, a final-year Information Technology engineering student who builds full-stack applications and documents them the way a production codebase should be documented. My work spans four domains — AI-enabled personal finance ([ZORO AI](https://github.com/dharmik-patel1511/zoro-ai#readme)), investor-startup collaboration ([InvestNook](https://github.com/dharmik-patel1511/Investnook#readme)), healthcare operations ([Hospital Management System](https://github.com/dharmik-patel1511/Hospital-Management-System#readme)), and enterprise HR workflows ([Human Resource Management System](https://github.com/dharmik-patel1511/Human-Resource-Management-System#readme)) — each built as a complete, database-driven system rather than a UI mockup.

Across these projects I've worked hands-on with responsive interface design, authentication, CRUD operations, REST APIs, and role-based workflow logic, while also handling the parts that don't show up in a demo — database schema design, documentation, and collaborating with a team under hackathon and coursework deadlines. I'm not claiming production-scale experience; what I can show is working software, built end to end, with the reasoning behind it written down.
